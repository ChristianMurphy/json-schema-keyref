lookup

  • lookup(queries: ISchemaDefinition, document: object): IQueryListing
    • Defined in index.ts:213

    Searches for a set of jsonpath queries.

    Parameters

    • queries: ISchemaDefinition

      identifier and jsonpath pairings to look up

    • document: object

      JSON document that will be searched

    Returns IQueryListing

    identifier and results pairings found in the document

    Example:

    var queries = [{"one": "$.a"}, {"two": "$.b"}];
var document = {"a": "something", "b": "something else"};

var result = lookup(queries, document);

console.log(result);
// {
//   "one": [{"path": ["$", "a"], "value": "something"}],
//   "two": [{"path": ["$", "b"], "value": "something else"}]
// }

    • Defined in index.ts:135

    Sorting mechanism for IQueryResult

    Parameters

    • a: IQueryResult

      first element

    • b: IQueryResult

      second element

    Returns number

    -1 for a less than b, 0 for a equals b, 1 for a greater than b

    Example:

    querySorter({"path": [], "value": "a"}, {"path": [], "value": "b"}); //=> -1
querySorter({"path": [], "value": true}, {"path": [], "value": true}); //=> 0
querySorter({"path": [], "value": 2}, {"path": [], "value": 1}); //=> 1
